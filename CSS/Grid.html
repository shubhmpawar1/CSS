Grid

CSS Grid is a 2D layout system in CSS that lets you arrange items in rows and columns easily. Unlike Flexbox (which is mainly 1D), Grid can handle both horizontal and vertical layout at the same time.

A. Container Property

1. Enable Grid


display: grid;          /* Block-level grid */
display: inline-grid;   /* Inline-level grid */

2. Define the Structure

i. grid-template-columns

grid-template-columns defines the number of columns in a CSS grid and their widths.

Sets column sizes

Example:


.container {
            display: grid;
            grid-template-columns: 100px  auto auto 100px;
            background-color: aquamarine;
            color: blue;
            gap: 10px;
        }

3. Control Gaps

i. gap

Sets both row and column gap

ii. row-gap

Space between rows

iii. column-gap

Space between columns

4. Alignment (Whole Grid)

i. justify-content

In CSS Grid, justify-content controls how the entire grid is positioned along the row (inline) axis inside the grid container when there’s extra horizontal space.

Values:

  • start → Aligns grid to the left.
  • end → Aligns grid to the right.
  • center → Centers the grid.
  • space-between → First and last tracks stick to edges; equal gaps between tracks.
  • space-around → Equal gaps around each track (half-gap at edges).
  • space-evenly → Equal gaps between and at edges.
