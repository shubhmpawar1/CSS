Grid

CSS Grid is a 2D layout system in CSS that lets you arrange items in rows and columns easily. Unlike Flexbox (which is mainly 1D), Grid can handle both horizontal and vertical layout at the same time.

A. Container Property

1. Enable Grid


display: grid;          /* Block-level grid */
display: inline-grid;   /* Inline-level grid */

2. Define the Structure

i. grid-template-columns

grid-template-columns defines the number of columns in a CSS grid and their widths.

Example:


.container {
  display: grid;
  grid-template-columns: 200px 1fr 1fr; /* Both axes */

}
