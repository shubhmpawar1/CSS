Size, Position & Lists

1. px – Pixels (Fixed Size)

  • Definition: A fixed unit representing a single dot on the screen.
  • Use Case: Best for precise control.
  • Responsive? ❌ No.
div {
  width: 200px; /* Always 200 pixels wide */
}
200px wide

2. % – Percentage of Parent Element

  • Definition: Relative to parent element size.
  • Use Case: Fluid, adaptable layouts.
  • Responsive? ✅ Yes.
.parent {
  width: 400px;
}

.child {
  width: 50%; /* 200px in this case */
}
Parent (400px)
Child (50% of parent)

3. em – Relative to Parent Font Size

  • Definition: Relative to parent font-size.
  • 1em = current font size.
  • Use Case: Good for scaling text, padding, margins.
  • Responsive? ✅ Depends on parent.
body {
  font-size: 16px;
}

p {
  font-size: 1.5em; /* 1.5 × 16px = 24px */
}
Parent font size (16px)
Child font size (1.5em)
