What Is Dom ?

DOM stands for Document Object Model.

It is a tree-like structure created by the browser when it loads an HTML page.

Simple Explanation:

When a web page loads:

  1. The browser reads your HTML.

  2. It creates a DOM — a tree of objects representing the structure of your page.
Example:

html code:


<html>
  <body>
    <h1>Hello</h1>
    <p>Welcome to my site</p>
  </body>
</html>

This becomes a DOM Tree like:


Document
 └── html
     └── body
         ├── h1 → "Hello"
         └── p → "Welcome to my site"
What is DOM in CSS?

When you write CSS, you're targeting elements in the DOM.

The browser uses the DOM Tree to figure out:

  • Which elements to style
  • What styles to apply
  • In what order (cascading)

So CSS applies styles based on the structure of the DOM.

What is a CSS Selector ?

A CSS selector is a pattern used to select HTML elements you want to style.

Basic Types of CSS Selectors:

Selector Targets Example CSS
Element All tags of that type <p> p { color: red; }
Class Elements with a class <div class="box"> .box { background: yellow; }
ID A unique element with an ID <h1 id="main"> #main { font-size: 24px; }
Universal All elements * * { margin: 0; }

CSS Rules Explained

A CSS rule defines how HTML elements should look. It consists of a selector and a declaration block.

Basic Structure: 

selector {
  property: value;
  property: value;
}
Example: 

p {
  color: blue;
  font-size: 16px;
}
  • Selector: p (targets all <p> tags)

  • Declarations:
    • color: blue; (makes text blue)

    • font-size: 16px; (sets text size)
Part Description
Selector Targets the HTML elements to style (p, .class, #id, etc.)
Property What you want to change (like color, background, font-size)
Value The setting for the property (red, 24px, center, etc.)
Declaration A pair of property and value, like color: red;
Declaration block All declarations within { }, separated by ;

2. Background Property

In CSS, the background property is a shorthand used to set multiple background-related values in a single line.

Syntax: 

background: [color] [image] [position] / [size] [repeat] [attachment] [origin] [clip];
Example: 

div {
  background: #f0f0f0; /* Just background color */
}

body {
  background: url("image.jpg") no-repeat center center / cover;
}

Other Related Background Properties:
Property What it Does Example
background-color Sets background color only background-color: yellow;
background-image Sets background image background-image: url("bg.jpg");
background-repeat Repeat image or not background-repeat: no-repeat;
background-position Position of image background-position: center top;
background-size Size of background image background-size: cover;
background-attachment Fixed or scroll with page background-attachment: fixed;

background-image:

Defines the image used as the background.

Css Syntax: 

background-image: url("image.jpg");

background-repeat:

The background-repeat property in CSS controls whether and how a background image repeats (tiles) within an element.

Css Syntax: 

/* 1. Repeats both horizontally and vertically */
body {
  background-image: url("bg.jpg");
  background-repeat: repeat;
}

/* 2. Repeats horizontally only */
body {
  background-image: url("bg.jpg");
  background-repeat: repeat-x;
}

/* 3. Repeats vertically only */
body {
  background-image: url("bg.jpg");
  background-repeat: repeat-y;
}

/* 4. No repeat at all */
body {
  background-image: url("bg.jpg");
  background-repeat: no-repeat;
}

/* 5. Repeats with space between images */
body {
  background-image: url("bg.jpg");
  background-repeat: space;
}

/* 6. Scales image to fit container without clipping */
body {
  background-image: url("bg.jpg");
  background-repeat: round;
}

background-size

The background-size property in CSS defines the size of the background image.

Syntax: 

/* background-size: auto; → Keeps the image's original size */
background-size: auto;

/* background-size: cover; → Stretches image to cover the element (may crop) */
background-size: cover;

/* background-size: contain; → Scales image to fit the element (no cropping) */
background-size: contain;

/* background-size: 200px 150px; → Custom width and height for image */
background-size: 200px 150px;

/* background-size: 100% auto; → Full width, height adjusts automatically */
background-size: 100% auto;

background-position

The background-position property in CSS sets the starting position of a background image within an element.

Syntax: 

background-position: x y;
  • x → horizontal position (left, center, right, or specific length/percentage)
  • y → vertical position (top, center, bottom, or specific length/percentage)

background-position: left top;        /* Top-left corner (default) */
background-position: center center;   /* Center of the element */
background-position: right bottom;    /* Bottom-right corner */
background-position: 50% 50%;         /* Same as center center */
background-position: 10px 20px;       /* 10px from the left, 20px from the top */
background-position: left 20px;       /* Left edge, 20px from the top */
background-position: 20% 80%;         /* 20% across and 80% down from the top-left corner */

background-attachment

The background-attachment property in CSS controls whether a background image scrolls with the page or stays fixed in place.

Syntax:

Chapter 3 [Css Box Model]

I. Css Box Model

The CSS Box Model describes how every HTML element is structured and how spacing and size are calculated.

Each element is like a box made up of four layers:

Diagram:
Box Model Layers:
  1. Content
    • The actual text, image, or element.
    • Set width & height here: width, height.
  2. Padding
    • Space inside the element, around the content.
    • Increases box size.
    • padding: 10px;
  3. Border
    • The border surrounding the padding.
    • You can style it with color, width, style.
    • border: 2px solid black;
  4. Margin
    • Space outside the element.
    • Creates space between elements.
    • margin: 20px;
Example: 

.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid green;
  margin: 15px;
}
  • Total Width = 200 + (20 × 2) + (5 × 2) = 250px
  • Total Height = 100 + (20 × 2) + (5 × 2) = 150px

Margins are outside, so they're not included in the total box size.

Shorthand Notion: 

/* 1 value → all sides */
margin: 10px;        /* top, right, bottom, left */
padding: 10px;

/* 2 values → top & bottom | left & right */
margin: 10px 20px;   /* 10px top & bottom, 20px left & right */
padding: 5px 15px;

/* 3 values → top | left & right | bottom */
margin: 10px 15px 20px;
padding: 5px 10px 15px;

/* 4 values → top | right | bottom | left */
margin: 10px 20px 30px 40px;
padding: 5px 10px 15px 20px;

Box-Sizing in CSS

The box-sizing property defines how the total width and height of an element is calculated.

Default Box Model (Without box-sizing)

Syntax:

box-sizing: content-box;

  • The width and height only include the content.
  • Padding and border are added outside this width and height.

Better Model (Common Practice)

Syntax:

box-sizing: border-box;

  • The width and height include padding and border.
  • This makes layout easier and more predictable.

Example:

Summary: box-sizing Comparison
Property .child1 (content-box) .child2 (border-box)
Width behavior Width excludes padding & border Width includes padding & border
Total size Larger than 200px Exactly 200px
Layout effect May overflow parent Stays inside parent

In CSS, the display property controls how an element is displayed on the page — it defines its layout behavior (e.g., block, inline)

Common display Values
Value Description

Font-family

The font-family property defines the typeface (font) used for text in an element.

Syntax:

font-family: "Font Name", fallback1, fallback2, ...;

You can list multiple fonts — the browser will use the first available one.

Example:

Html Font-familiy

<p style="font-family: 'Roboto', 'Helvetica Neue', Arial, sans-serif;">
  Your text goes here.
</p>

If Roboto is not available, it tries the next one (Helvetica Neue, then Arial, then default sans-serif).

Google Font

<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

Web Safe Font

Web safe fonts are fonts that are universally installed across most operating systems and browsers. These fonts ensure that your website’s text looks consistent for all users.

Best Practice

font-family: Arial, Helvetica, sans-serif;
font-family: 'Times New Roman', Times, serif;
font-family: 'Courier New', Courier, monospace;

This ensures:

  • If the first font is missing, the browser tries the next one.
  • Ends with a generic family like sans-serif, serif, or monospace.
