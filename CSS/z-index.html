z-index

  • The z-index property controls the stacking order of elements on a web page along the z-axis (depth).
  • It determines which elements appear on top of others when they overlap.

How does it work?

  • Elements with a higher z-index value appear in front of elements with a lower value.
  • Only works on elements with a position value of relative, absolute, fixed, or sticky.
  • Default z-index is auto (usually 0).

Important notes

  • z-index only affects stacking of elements that are positioned (position not static).
  • Negative z-index can send elements behind others.
  • If two positioned elements have the same z-index, the one later in the HTML code stacks on top.
