min-max (height-width)

Min-max properties in CSS are settings that define the smallest (min-) and largest (max-) size an element can have, for either width or height.

1. min-width / max-width

  • min-width → The smallest width an element can be.
  • max-width → The largest width an element can be.
  • Useful for responsive layouts — prevents elements from becoming too narrow or too wide.
.card {
  min-width: 200px;  /* can't be smaller than 200px */
  max-width: 600px;  /* can't be larger than 600px */
}

Example:

min-width: 200px; max-width: 600px;

2. min-height / max-height

  • min-height → The smallest height an element can have.
  • max-height → The tallest an element can be.
  • Common for scrollable areas — lets content grow up to a limit before scrolling.

Example:

.chat-box {
  min-height: 100px;
  max-height: 200px;
  overflow-y: auto; /* adds scroll if content exceeds max-height */
}

Why They’re Useful

  • Responsive design: Works great with %, vw, vh, or clamp() values.
  • Prevent layout breaking: Stops buttons, text boxes, and images from becoming unusably small or awkwardly large.
  • Control over resizing: Especially important in flexbox or grid layouts.
