The Float Property

The float property in CSS is used to position an element to the left or right within its container, allowing other content (like text) to flow around it. It’s often used for creating layouts or for wrapping text around images.

Syntax:

selector {
  float: none | left | right | inline-start | inline-end;
}

Values:

  • none (default): The element does not float; it stays in the normal flow.
  • left: The element floats to the left of its container.
  • right: The element floats to the right of its container.

The clear property

In CSS, the clear property controls whether an element is allowed to sit beside floating elements (float) or if it should be moved down below them.

It’s typically used to “clear” space under floated elements so that the next block element starts on a new line without wrapping around them.

Syntax:


element {
  clear: none | left | right | both | inline-start | inline-end;
}

Values:

  • none — Default. Allows the element to be next to floated elements.
  • left — Moves the element below any floated elements on the left.
  • right — Moves the element below any floated elements on the right.
  • both — Moves the element below floated elements on both sides.

Example:


<style>
.box {
  width: 100px;
  height: 100px;
  background: lightblue;
  float: left;
  margin: 10px;
}
.clearfix {
  clear: both;
}
</style>

<div class="clear-box">1</div>
<div class="clear-box">2</div>
<p class="clearfix">This paragraph starts below the floated boxes.</p>

What happens:

  • Without clear: both;, the paragraph would wrap around the floated boxes.
  • With clear: both;, it jumps below them.

Tip: In modern layouts, floats are less common for positioning (we use Flexbox/Grid now), but clear is still useful when working with older float-based designs or for clearing images inside text.

Flexbox

Flexbox (short for “Flexible Box Layout”) is a CSS layout system designed to arrange elements in a row or column, and distribute space between them without having to use floats, tables, or complicated positioning tricks.

Think of it as a “smart box” that knows how to arrange its children neatly, even if their sizes vary.

How to activate Flexbox:

You turn any element into a flex container by giving it:


.container {
  display: flex;
}

Now all direct children of .container become flex items.

Main concepts:

Flexbox works along two axes:

  • Main axis → Direction in which items are laid out (row or column).
  • Cross axis → Perpendicular to the main axis.
properties On the Container
Property Purpose
flex-direction Sets main axis direction: row (default), row-reverse, column, column-reverse
justify-content Aligns items along the main axis: flex-start, flex-end, center, space-between, space-around, space-evenly
align-items Aligns items along the cross axis (within each row/column): flex-start, flex-end, center, stretch, baseline
align-content Aligns multiple rows (or columns, if flex-direction: column) along the cross axis when there’s extra space: flex-start, flex-end, center, space-between, space-around, stretch
flex-wrap Controls wrapping: nowrap (default), wrap, wrap-reverse
gap Adds space between items without using margins
properties On the Items
Property Purpose
order Controls the order in which a flex item appears; lower values appear first (default is 0)
flex-grow How much the item can grow relative to others
flex-shrink How much the item can shrink relative to others
flex-basis Initial size before growing/shrinking
flex Shorthand for grow, shrink, basis (e.g. flex: 1 0 200px)
align-self Overrides container’s align-items for a single item
