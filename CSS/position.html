CSS Position.

In CSS, the position property specifies how an element is positioned in a document. There are several types of positioning:

1. static (default)

  • The element follows the normal flow of the page.
  • top, left, right, bottom properties have no effect.

2. relative

  • The element is positioned relative to its normal position.
  • You can use top, left, right, bottom to move it relative to where it would normally be.
position: relative;
top: 20px;  /* moves the element 10px down from its normal spot */
left: 30px; /* moves the element 5px to the right */

Example:

This Original Position
after top: 20px; left: 30px;

3. absolute

  • The element is taken out of the normal page flow. Other elements will behave as if it’s not there.
  • You can place the element exactly where you want by using top, left, right, and bottom values.
  • It moves relative to the closest parent element that has a position other than static. If no parent has a position set, it will position itself relative to the whole page (the browser window).
position: absolute;
top: 20px;
right: 10px;

Example:

The element is taken out of the normal page flow.

4. fixed

  • The element is positioned relative to the viewport.
  • It stays in the same place even when the page is scrolled.
  • Removed from normal document flow.

position: fixed;
bottom: 0;
right: 0;

Example:

position: fixed; bottom: 0; right: 0;

5. sticky

  • The element acts like it’s normally positioned (like relative) at first.
  • But when you scroll the page and the element reaches a certain spot (like the top of the screen), it sticks there — it stays fixed in place.
  • Once you scroll back up, it goes back to normal flow.
position: sticky;
top: 0;  /* sticks to the top when you scroll */
position: sticky; bottom: 0; /* sticks to the bottom when you scroll */
